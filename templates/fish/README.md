# Advent of Code in fish

## Usage

```sh
./run.fish     # run with input.txt
./sample.fish  # run with sample.txt
./test.fish    # run tests
```

## Watch Mode

```sh
fd | entr -cc ./run.fish     # run with input.txt
fd | entr -cc ./sample.fish  # run with sample.txt
fd | entr -cc ./test.fish    # run tests
```
