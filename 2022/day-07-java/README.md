# Advent of Code 2022 Day 07 (Java/JUnit5/Gradle)

Based on `junit5-jupiter-starter-gradle` from
[junit5-samples](https://github.com/junit-team/junit5-samples).

## Usage

```sh
./gradlew test
./gradlew run
```

## Info

The `junit5-jupiter-starter-gradle` project demonstrates how to run tests based
on JUnit Jupiter using [Gradle's native JUnit Platform support], Gradle's Groovy
DSL and code and tests written in Java.

[Gradle's native JUnit Platform support]: https://docs.gradle.org/current/userguide/java_testing.html#using_junit5
